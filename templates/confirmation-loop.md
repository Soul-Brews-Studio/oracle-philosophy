# Human Confirmation Loop Template

Use this before any permanent action (commit, publish, delete).

---

## Before Confirming

Ask yourself:

1. **Is this what I actually want?**
   - [ ] Yes, this reflects my intention
   - [ ] No, needs adjustment

2. **Am I missing anything?**
   - [ ] Nothing missing
   - [ ] Missing: ___________

3. **Ready to commit or edit first?**
   - [ ] Ready to commit
   - [ ] Need to edit

## The Confirmation Prompt

When working with AI, always include:

```
**Before we proceed:**
- Is this okay?
- Missing anything?

Ready to commit or edit first?
```
